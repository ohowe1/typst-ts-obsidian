# Typst.ts Obsidian

Typst rendering support for Obsidian using [Typst.ts](https://github.com/Myriad-Dreamin/typst.ts).

To use it, simply write Typst math inside `$...$` for inline math or `$$...$$` for block math in your Obsidian notes.

Please report any bugs on the [GitHub Issues page](https://github.com/ohowe1/typst-ts-obsidian/issues).
